# 概述

## 解释

超文本标记语言（Hypertext Markup Language）

* 超：通过超链接相互跳转
* 文本：文本格式
* 标记：使用标记
* 语言：解释性语言

## 工具

* 文本编辑器：Visiual Studio Code
* 浏览器：Chrome
* 本地Web服务器：http-server

## 运行

### 本地

* 新建HTML文件
* 开启本地Web服务器
* 通过浏览器访问HTML文件

### 在线

使用[codepen](https://codepen.io), [jsbin](http://jsbin.com/?html,output), [jsfiddle](https://jsfiddle.net/)等在线工具进行在线编辑和实时预览

### 服务器

* 将本地代码发布到服务器上
* 打开相应的网站浏览网页
