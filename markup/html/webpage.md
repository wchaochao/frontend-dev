# 网页

在浏览器上显示的`HTML`文档

## 编码

`<meta>`中的编码要和文件编码一致，推荐使用`UTF-8`

## 兼容性

不同浏览器显示网页时可能会有些细微的差别

主流浏览器：`IE`, `Chrome`, `Firefox`, `Opera`, `Safari`

## 空格

`html`文档中所有连续的空格或空行都会被算作一个空格显示

## 转义

`html`文档中的以下字符在网页中显示时会被转义

```
&amp; -> &
&nbsp; -> 空格
&lt; -> <
&gt; -> >
&quot; -> "
&cent; -> ￠
&pound; -> £
&yen; -> ¥
&euro; -> €
&sect; -> §
&copy; -> ©
&reg; -> ®
&trade; -> ™
&times; -> ×
&divide; -> ÷
```
